# Kereki-Madárpark-vizsgaremek
A Kereki madárpark weboldala. 

A projektünk a Kereki Madárpark elavult weboldalát kíván korszerűbbé és felhasználó barátabbá tenni.
A weboldal az oldallra látogatóknak, a madarak megvétele íránt érdeklődőknek, illetve az ott dolgozóknak található egy ADMIN felület is.
